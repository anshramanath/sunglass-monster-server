# Sunglass Monster Server

Next.js API server for the Sunglass Monster multi-brand sunglasses platform.

## Getting Started

```bash
npm run dev
```

Server runs at [http://localhost:3000](http://localhost:3000).

## API

See [API.md](API.md) for the full public API contract.
